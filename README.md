# Customer Behavior Dashboard

## 📊 Project Overview
This project is a Power BI dashboard designed to analyze customer shopping behavior and provide insights into customer demographics, purchasing patterns, and subscription trends.

The dashboard helps businesses understand customer preferences and make data-driven decisions.

---

## 🚀 Features

- Customer demographic analysis
- Age group distribution
- Subscription status analysis
- Customer behavior insights
- Interactive Power BI visualizations
- Data-driven business recommendations
